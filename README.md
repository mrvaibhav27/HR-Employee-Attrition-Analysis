# HR-Employee-Attrition-Analysis

## Project Overview
Employee attrition analysis using Python and Power BI to identify key factors affecting employee turnover and improve HR retention strategies.

## Objectives
- Analyze employee attrition trends
- Identify factors affecting attrition such as job satisfaction, salary, work-life balance, overtime, and promotions
- Support HR teams with data-driven insights

## Dataset
- 1,470 employee records
- Features include age, department, job role, salary, job satisfaction, overtime, years since last promotion, etc.

## Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization: Matplotlib, Seaborn
- Power BI (Interactive Dashboard)
- Scikit-learn (basic analysis)

## Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Attrition trend analysis by department and job role
- Visualization of retention and turnover patterns

## Dashboard
Developed an interactive Power BI dashboard to visualize:
- Attrition rate
- Job satisfaction trends
- Department-wise employee turnover
- Key HR KPIs

## Key Insights
- Identified departments with high attrition rates
- Highlighted impact of overtime and low job satisfaction on employee turnover
- Provided recommendations to improve retention
